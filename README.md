<!-- ═══════════════════ ABOUT ═══════════════════ -->

<img src="https://readme-typing-svg.demolab.com?font=Playfair+Display&weight=600&size=28&duration=2000&pause=3000&color=C9A34F&vCenter=true&width=300&height=50&lines=%E2%9C%A6+About" alt="About Section"/>

<img align="right" width="300" src="https://raw.githubusercontent.com/rodrigograca31/rodrigograca31/master/matrix.svg" alt="Matrix animation"/>

**Interactive Showcase Studio** is an immersive, scroll-driven luxury fashion showcase built as a modern web experience. It features the fictional brand **Maison Éclat** — a haute couture house — and demonstrates advanced animation techniques, parallax scrolling, clip-path mask reveals, and glassmorphism design patterns.

> <img src="https://media.giphy.com/media/VgCDAzcKvsR6OM0uWg/giphy.gif" width="20"/> *"Fashion is the armor to survive the reality of everyday life."*
>
> — Maison Éclat, Winter 2026

<!-- ═══════════════════ ANIMATION HIGHLIGHTS ═══════════════════ -->

<img src="https://readme-typing-svg.demolab.com?font=Playfair+Display&weight=600&size=28&duration=2000&pause=3000&color=C9A34F&vCenter=true&width=550&height=50&lines=%E2%9C%A6+Animation+Highlights" alt="Animation Section"/>

<br/>

<details>
<summary><img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="18"/> <b>Scroll-Driven Text Reveal</b></summary>
<br/>
<div align="center">
<img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&size=12&duration=1500&pause=500&color=50FA7B&center=true&vCenter=true&repeat=true&width=500&height=20&lines=useScroll()++→++useTransform()++→++Word+Animation" alt="scroll reveal"/>
</div>
<br/>
Words animate individually based on scroll position using <code>useScroll</code> and <code>useTransform</code> from Framer Motion — each word fades in and slides up as the user scrolls through the section.
</details>

<details>
<summary><img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="18"/> <b>Clip-Path Mask System</b></summary>
<br/>
<div align="center">

| Mask Type | Preview |
|:---------:|:-------:|
| `center` | ◻️ → ⬜ |
| `circle` | ⚫ → ⬤ |
| `diamond` | 🔹 → 🔷 |
| `left` / `right` | ◀️ ▶️ |
| `top` / `bottom` | 🔼 🔽 |

</div>
<br/>
A flexible <code>ClipReveal</code> component supports 7 mask types — all driven by scroll progress with eased timing.
</details>

<details>
<summary><img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="18"/> <b>Parallax Image Reveals</b></summary>
<br/>
<div align="center">
<img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&size=12&duration=1500&pause=500&color=BD93F9&center=true&vCenter=true&repeat=true&width=450&height=20&lines=scale(1.35x)+→+clipPath(open)+→+parallax(y)" alt="parallax"/>
</div>
<br/>
Images scale down from 1.35x while the clip-path opens, combined with vertical parallax offset — creating a cinematic "window opening" effect as elements enter the viewport.
</details>

<details>
<summary><img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="18"/> <b>Horizontal Scroll Carousel</b></summary>
<br/>
<div align="center">
<img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&size=12&duration=1500&pause=500&color=FF79C6&center=true&vCenter=true&repeat=true&width=450&height=20&lines=◁+Card+1+|+Card+2+|+Card+3+|+Card+4+▷;++Card+1+|+Card+2+|+Card+3+|+Card+4+◁" alt="carousel"/>
</div>
<br/>
The Collections section uses a pinned/sticky container with scroll-linked horizontal translation. Individual cards scale and fade based on their position in the scroll timeline.
</details>
